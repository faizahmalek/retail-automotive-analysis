# Retail Sales Performance Analysis (2003–2005)

## Project Objective
This project analyzes historical retail sales data for a global automotive products company to support strategic decisions on product focus, regional performance, and customer retention.

## Dataset
- 2,823 sales transactions
- Time period: 2003–2005
- Products: Classic cars, motorcycles, planes, ships, trains, trucks
- Source: Sample Sales Data (Kaggle)

## Tools Used
- Excel  – data exploration & cleaning
- SQLite – SQL analysis
- Power BI – data visualization & dashboard
- PowerPoint – business presentation

## Project Workflow
1. Data exploration and quality assessment
2. Data cleaning and preparation
3. SQL-based sales analysis
4. Dashboard creation and visualization
5. Business insights and recommendations

## Key Business Questions
- Which product lines generate the highest revenue?
- Which countries and regions perform best?
- Are there seasonal sales patterns?
- Who are the top customers?
- What is the order cancellation rate?

## Key Insights
- Revenue is concentrated in a few product lines
- Certain regions consistently outperform others
- Sales show clear seasonal trends
- A small group of customers contributes a large share of revenue

## Deliverables
- Cleaned dataset
- SQL scripts and query results
- Interactive Power BI dashboard
- Business report and presentation
